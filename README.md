# Веб-застосунок «Реєстрація на конференцію» (Python + FastAPI)

Проєкт виконано в межах проходження виробничої практики. Застосунок розроблено на базі асинхронного фреймворку FastAPI з використанням ORM SQLAlchemy для взаємодії з базою даних та шаблонізатора Jinja2 для рендерингу веб-сторінок.

## Функціональні можливості та архітектура
Сайт складається з 4 основних сторінок, об'єднаних спільною адаптивною версткою на Bootstrap 5 через єдиний базовий шаблон:
1. **Форма реєстрації:** приймає ім'я, прізвище, стать, національність, організацію, посаду, дату народження, email та пароль з підтвердженням.
2. **Форма входу:** аутентифікація користувачів з випуском токенів доступу JWT, які надійно зберігаються у HTTPOnly Cookies браузера клієнта.
3. **Список зареєстрованих учасників:** захищена сторінка, доступна лише авторизованим користувачам. Виводить порядковий номер, об'єднані ім'я та прізвище в одній клітинці, назву організації та email.
4. **Форма редагування даних:** захищена сторінка, яка дозволяє змінювати персональні дані, організацію та посаду. Реалізовано жорстку ізоляцію прав доступу на стороні сервера — користувач може редагувати виключно власні дані (для чужих профілів кнопка редагування блокується).

### Безпека та валідація даних
* **Захист паролів:** реалізовано за допомогою бібліотеки passlib (алгоритм pbkdf2_sha256). Паролі в базі даних зберігаються виключно у вигляді безпечних криптографічних хешів.
* **Валідація форм:** подвійна перевірка даних. На стороні клієнта використовуються стандартні HTML5 атрибути required, type="email" та type="date". На стороні сервера впроваджено перевірку унікальності Email перед реєстрацією, валідацію збігу паролів та обмеження їхньої мінімальної довжини у 6 символів.

## Інструкція з локального розгортання

# 1. Перехід у папку проєкту
cd RegistrationConf
# 2. Створення віртуального оточення
python -m venv .venv
# 3. Активація (для Windows PowerShell)
.venv\Scripts\Activate.ps1
# 4. Альтернативна активація (якщо PowerShell блокує скрипти, виконайте це для CMD)
& .venv/Scripts/activate.bat
# 5. Встановлення всіх необхідних пакетів і залежностей
pip install fastapi uvicorn sqlalchemy alembic jinja2 python-multipart pyjwt passlib
# 6. Ініціалізація бази даних (виконання міграцій)
alembic upgrade head
# 7. Наповнення бази даних початковими довідниками
python seed.py
# 8. Запуск локального веб-сервера розробки
uvicorn main:app --reload
